{% block title %}About Paper to Speech{% endblock %}

This application allows you to easily convert various file formats to MP3 format.

Simply drag and drop your file onto the drop area on the home page, and the converted file will be automatically downloaded.

Supported file formats include:

  • PDF documents (.pdf)
  • Text files (.txt)
  • Images (.png, .jpg, .jpeg, .gif)
  • Word documents (.docx)

Note: Files are automatically deleted from the server after download for privacy.
